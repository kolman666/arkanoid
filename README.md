# Arkanoid

Клон аркады **Arkanoid** (Famicom, 1986) на чистом HTML5 + CSS3 + JavaScript.

## Запуск

Открыть `index.html` в браузере. Никакой сборки не требуется.

## Управление

- **← / →** или **мышь** — движение ракетки
- **Пробел / клик** — запуск мяча

## Структура

```
index.html      разметка
css/style.css   стили
js/game.js      игровая логика
```
