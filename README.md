# Arkanoid — тестовое задание (Peppers)

Клон аркады **Arkanoid** (Famicom, 1986) на чистом **HTML5 + CSS3 + JavaScript**,
без фреймворков и сборки. Вся отрисовка — на `<canvas>`.

## Запуск

Открыть `index.html` в браузере — больше ничего не нужно.

> Некоторые браузеры блокируют звук до первого клика/нажатия клавиши — это нормально,
> звук включится после старта игры.

## Управление

| Действие | Клавиши |
|----------|---------|
| Движение ракетки | **← →** или **мышь** |
| Запуск мяча / выстрел лазера | **Пробел** или **клик** |
| Новая игра (после Game Over / победы) | **Пробел** |

## Возможности

- Физика мяча с изменением угла отскока в зависимости от точки касания ракетки
- 4 уникальные схемы уровней (уровни зацикливаются, скорость растёт)
- Типы кирпичей: **цветные** (1 удар), **серебряные** (2 удара), **золотые** (неразрушимые)
- 6 бонусов-капсул в духе оригинала:
  - **E** — расширение ракетки
  - **C** — «ловушка» (мяч прилипает к ракетке)
  - **S** — замедление мяча
  - **D** — тройной мяч
  - **L** — лазер (стрельба по кирпичам)
  - **P** — дополнительная жизнь
- Очки, жизни, номера уровней (HUD)
- Стартовый экран с **обучением**: управление и легенда бонусов
- Анимация: частицы-осколки, след мяча, мерцающие звёзды, пульсация UI
- Звук через **WebAudio** (без внешних файлов)

## Структура проекта

```
index.html      разметка страницы
css/style.css   стили страницы и контейнера
js/game.js      вся игровая логика (изолирована в IIFE + класс Arkanoid)
```

Весь функционал завёрнут в самовызывающуюся функцию и класс `Arkanoid`,
глобальные имена не создаются — код не пересекается с чужими скриптами на странице.

## Архитектура кода

Игра разбита на небольшие изолированные классы:

- `Arkanoid` — игровой цикл, состояния (`menu / ready / playing / gameover / win`), рендер
- `Paddle` — ракетка (Vaus): движение, режимы Catch и Laser, анимация ширины
- `Ball` — мяч: движение, отскоки, след
- `Brick` — кирпич: прочность, типы, вспышка при попадании
- `Capsule` — падающие бонусы
- `Laser` — лазерные выстрелы
- `Particle` — осколки кирпичей
- `Sound` — звук на WebAudio
- `Input` — клавиатура и мышь
